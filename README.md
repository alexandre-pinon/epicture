# Epicture

Made by ストレモン .<br>
The goal of this project is to use the Imgur API.<br>

Written in [React-Native](https://reactnative.dev/).

## List of commands:

```sh
$ npm install
```
## Build and install on your device/emulator.

```sh
$ npm run android
```
